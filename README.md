# 🏦 Smart Banking System – C++ Console Application

The Smart Banking System is a **C++ console-based client management application** developed as part of the  
[AbouHadhood Platform](https://programmingadvices.com/) training track with Dr. Mohamed Abou Hadhoud.

The project was built incrementally through multiple versions, with each version focusing on improving
**functionality, structure, and code clarity**.

---

## 📁 Project Versions Overview

| Version | Description | Key Features |
|--------|-------------|--------------|
| **Bank v1** | Initial version implementing the core banking client management logic. | Modular design, structs & enums, file-based data storage, clean console output. |
| **Bank v1.1 (Extension)** | Extended version adding account balance handling and transactions. | Deposit & withdraw operations, balance display, total bank balance overview. |

---

## 📖 About the Project

This project simulates a **basic real-world banking system** using a console interface.

The focus is not only on features, but also on **clean code principles**:
- Clear separation of responsibilities
- Small, well-named functions
- Maintainable and readable structure
- File-based persistence for client data

The system is designed to be easy to extend and understand, making it a strong foundation for more advanced banking features in future versions.
