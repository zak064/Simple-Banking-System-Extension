## 📖 About the Project

**Smart Banking System** is a **C++ console-based client management application** that simulates a simple banking system.

The project was built step by step through multiple versions. Each version improves the functionality, structure, and code clarity.

The main goal of this project is to practice C++ fundamentals, file handling, modular programming, and clean code principles.

---

## 📁 Project Versions Overview

| Version | Description | Key Features |
|--------|-------------|--------------|
| **Bank v1** | Initial version with core banking client management features | Add, show, search, update, and delete clients |
| **Bank v1.1** | Extended version with account balance and transaction features | Deposit, withdraw, balance display, and total bank balance overview |
